# 我的Neovim配置！

**请您仔细阅读本说明文件后使用配置**

## 安装后, 您需要:

- [ ] 安装 `pynvim` (pip)
- [ ] 安装 `nodejs`
- [ ] 安装 nerd-fonts (此选项为可选，但使用nerd-font后，看起来真的不错)

## 安装后，你可能需要做:

- [ ] 执行 `:checkhealth`
- [ ] 确保你有Python
- [ ] 编辑 `special_cofnig.vim` 进行配置
- [ ] 执行 `pip3 install flake8` (用于代码检查)
- [ ] 安装 `ctags` 用于显示函数/类/变量列表
- [ ] 安装 `figlet`
